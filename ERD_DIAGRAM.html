📋 Chi tiết các Bảng Dữ liệu

Bảng COURSES

Cột Kiểu dữ liệu Ràng buộc Mô tả
id INT PK, Auto increment Khóa chính
name VARCHAR(255) NOT NULL Tên khóa học
slug VARCHAR(255) NOT NULL, UNIQUE URL-friendly tên (web-development)
price DECIMAL(10,2) NOT NULL Giá khóa học
description TEXT NULL Mô tả chi tiết
image VARCHAR(255) NULL Đường dẫn ảnh đại diện
status ENUM NOT NULL, DEFAULT='draft' Trạng thái: draft, published
deleted_at TIMESTAMP NULL (Soft Delete) Thời gian xóa mềm
created_at TIMESTAMP NOT NULL Thời gian tạo
updated_at TIMESTAMP NOT NULL Thời gian cập nhật

Bảng LESSONS

Cột Kiểu dữ liệu Ràng buộc Mô tả
id INT PK, Auto increment Khóa chính
course_id INT FK, NOT NULL Khóa ngoại tới COURSES
title VARCHAR(255) NOT NULL Tên bài học
content TEXT NOT NULL Nội dung bài học
video_url VARCHAR(255) NULL Link video (YouTube, ...)
order INT NOT NULL Thứ tự hiển thị bài học
created_at TIMESTAMP NOT NULL Thời gian tạo
updated_at TIMESTAMP NOT NULL Thời gian cập nhật

Bảng STUDENTS

Cột Kiểu dữ liệu Ràng buộc Mô tả
id INT PK, Auto increment Khóa chính
name VARCHAR(255) NOT NULL Tên học viên
email VARCHAR(255) NOT NULL, UNIQUE Email học viên
created_at TIMESTAMP NOT NULL Thời gian tạo
updated_at TIMESTAMP NOT NULL Thời gian cập nhật

Bảng ENROLLMENTS (Pivot Table - N:M)

Cột Kiểu dữ liệu Ràng buộc Mô tả
id INT PK, Auto increment Khóa chính
course_id INT FK, NOT NULL Khóa ngoại tới COURSES
student_id INT FK, NOT NULL Khóa ngoại tới STUDENTS
created_at TIMESTAMP NOT NULL Thời gian đăng ký
updated_at TIMESTAMP NOT NULL Thời gian cập nhật

🔗 Các Quan hệ Dữ liệu

Quan hệ Loại Mô tả
Courses ← → Lessons 1:N Một khóa học có nhiều bài học
Courses ← → Enrollments 1:N Một khóa học có nhiều đơn đăng ký
Students ← → Enrollments 1:N Một học viên có nhiều đơn đăng ký
Courses ← → Students N:M Một khóa học có nhiều học viên, một học viên học nhiều khóa (qua bảng ENROLLMENTS)
Users ← → Courses 1:N Một người dùng (giáo viên) tạo nhiều khóa học (tương lai)
